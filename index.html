Installing Astropy on Mac OS X

The astropy package can easily be installed using:

pip install astropy

Alternatively, the astropy package is also included by default in the Anaconda Python Distribution and Enthought Canopy, which provide an easy way to get set up with a scientific Python distribution. Astropy can also be installed with MacPorts.

Installing Astropy on Windows

There are a number of options for installing the astropy package on Windows. If you are not comfortable with downloading the source code from PyPI and installing it from source, there are also binary installers available on PyPI. Alternatively, astropy is included by default in the Anaconda Python Distribution and Enthought Canopy, both of which provide an easy way to get set up with a scientific Python distribution.

Getting Astropy's Source Code

The source code for the current release of astropy can always be downloaded from Astropy's PyPI page

If instead you want the current developer version, it is best obtained from github. If you have git installed you can get it by doing: 

git clone https://github.com/astropy/astropy.git

Or if you don't have git, you can download a zip file of the in-development version.

Do none of the above instructions work for your system, or do you need more detailed instructions? Check out the installation instructions in our documentation.
